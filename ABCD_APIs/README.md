# ABCD Strapi APIs

CURL GET requests for ABCD Portal APIs.

## Usage

```bash
# Copy the curl command from respective file and execute
curl -X GET "https://abcduat1.abcscuat.com/strapi/api/..."
```

## APIs

### digimetal

| API | Description | File |
|-----|-------------|------|
| digimetal-csp-configs | Content Security Policy configuration for DigiMetal products | [csp-configs.md](digimetal/csp-configs.md) |
| digimetal-dashboard-pages | Dashboard page content for DigiMetal products | [dashboard-pages.md](digimetal/dashboard-pages.md) |
| digimetal-journey-pages | Journey flow pages for DigiMetal products | [journey-pages.md](digimetal/journey-pages.md) |
| digimetal-landing-pages | Landing page content for DigiMetal products | [landing-pages.md](digimetal/landing-pages.md) |
| digimetal-channel-mappings | Channel mapping configuration | [channel-mapping.md](digimetal/channel-mapping.md) |
| digimetal-navigations | Navigation configuration | [navigation.md](digimetal/navigation.md) |
| digimetal-partner-configs | Partner-specific configuration | [partner-config.md](digimetal/partner-config.md) |
